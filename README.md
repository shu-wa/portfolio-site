# Portfolio Site

## 概要
就職活動で自身の制作物を紹介するために作成したポートフォリオサイトです。
Next.js、TypeScript、Tailwind CSSを用いて実装し、AWS Amplify Hostingで公開しています。

## 公開URL
https://main.d217a718xi2gst.amplifyapp.com/

## 使用技術
- Next.js
- TypeScript
- Tailwind CSS
- Git / GitHub
- AWS Amplify Hosting

## 主な機能
- 自己紹介表示
- 制作物一覧表示
- 制作物詳細ページ
- レスポンシブ対応

## 今後の予定
- API Gateway + LambdaによるAPI実装
- DynamoDBによる作品データ管理
- Cognitoによる管理者ログイン
- 管理画面からの作品追加・編集
